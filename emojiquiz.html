Emoji Riddle Quiz App

Project Overview

This app is a fun and interactive quiz game where users try to guess movies, foods, drinks, places, and historical landmarks based on emojis. Each question challenges the user to observe the image carefully and pick the correct answer from multiple-choice options.

Technologies Used

  • React Native
  • CSS

Features

  • Randomized Questions: Each quiz presents a different set of 6 questions randomly selected from a pool, keeping the quiz fresh and replayable.
  • Immediate Feedback with Hints: When a user selects a wrong answer, a hint appears to guide them toward the correct answer.
  • Score Tracking & Alerts: After submitting, the app calculates the score and displays a motivational message based on performance.
  • Prevent Incomplete Submission: Users cannot submit the quiz without answering all questions.
  • Restart Quiz Option: Users can restart the quiz at any time to try again with a new random set of questions.
